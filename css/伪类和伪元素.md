# 伪类和伪元素

标签（空格分隔）： css

---
可以参考MDN对[伪类][1]和[伪元素][2]的定义。

伪类是元素处于某个状态时，为它添加样式。操作对象是文档树中已有的元素。
伪元素是在文档树中不在的元素，并为它添加样式。而伪元素是创建一个文档外的元素。
伪类和伪元素的根本区别在于：它们是否创造了新的元素。
