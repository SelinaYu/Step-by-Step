# 冒泡排序

```javascript
function sort(arr){
  for(let i = 0;i < arr.length;i++){
      for(let j = 0;j < arr.length - 1 - i;j++){
        if(arr[j] > arr[j+1]){
            let temp = arr[j];
            arr[j] = arr[j+1];
            arr[j+1] = temp;
        }
      }
  }
  return arr;
}
```

## 思路过程

1) 比较相邻两个元素，如果前一个比后一个大，则交换位置
2) 对每个相邻元素做相同操作，直至遍历一轮元素，则能得到一个最大的元素
3) 针对所有元素重复以上步骤，除了已经排序好的最大元素

由于两遍循环，最坏的情况，时间复杂度为O(n2),最好的情况时间复杂度为O(n),属于稳定排序，平均时间复杂度为O(n2)
