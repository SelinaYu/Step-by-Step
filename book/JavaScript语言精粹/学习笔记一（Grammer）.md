# 学习笔记一（Grammer）

标签（空格分隔）： JavaScript语言精粹

---

(一) 避免使用`/* */`注释，而用`//`注释代替它。因为`/**/`可能会出现在正则表达式字面量里，造成语法错误。  
```
/*
  var rm_a = /a*/.match(s);
*/
```

(二) 转义字符用来把正常情况下不被允许的字符插入到字符串中，比如**反斜杠，引号，和控制字符**。其中，`\u`约定用来指定数字字符编码。

(三) 下面列出的值被当做假,其他的值都是真：

- false
- null
- undefined
- 空字符串' '
- 数字 0
- 数字 NaN
